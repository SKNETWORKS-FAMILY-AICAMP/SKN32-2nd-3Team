# 고객 이탈 예측 AI 모델 생성을 위한 데이터 전처리 결과서

## 1. 개요
* **프로젝트명:** KISDI 통계 데이터 기반 OTT 서비스 고객 이탈(Churn) 예측 AI 모델 개발
* **프로젝트 목적:** * KISDI(정보통신정책연구원)의 다개년 방송통신통계 원시자료를 기반으로 고객의 OTT 이용 행태 변화를 추적합니다.
  * 시계열 피처 엔지니어링을 통해 고객의 이탈 징후(이용 시간/비용 감소 등)를 사전 감지하는 머신러닝 예측 모델의 데이터 기반을 구축합니다.
* **원천 데이터 출처:** KISDI KISSTAT 방송통신통계포털 미디어패널 원시자료 (2021년 ~ 2025년)

---

## 2. 데이터 수집 및 구축 현황

### 2.1 원천 데이터 (Raw Data) 현황
다개년(2021년~2025년) 패널 조사 결과로, 매년 약 500여 명의 응답자와 최대 10,000개에 달하는 방대한 설문 문항(열)으로 구성되어 있습니다.
* `p21v32_KMP.csv` : 523행 × 10,155열
* `p22v32_KMP.csv` : 542행 × 9,942열
* `p23v32_KMP.csv` : 518행 × 9,758열
* `p24v32_KMP.csv` : 522행 × 8,694열
* `p25v32_KMP.csv` : 532행 × 8,412열

### 2.2 중간 정제 데이터 현황
* **AI 정제 마스터 데이터 (`final_churn_master_dataset_21_25.csv`)**: 203행 × 38,546열
  * 5개년 데이터 중 연속적으로 추적된 핵심 패널(203명)을 식별하고 다개년 데이터를 가로로 결합(Wide Format)하여 통합 마스터 데이터셋을 생성했습니다.
* **도메인 특화 수동 정제 데이터**: 이탈 예측의 핵심 요인인 OTT 이용 행태(이용 여부, 이용 시간, 지출 비용)를 유저별·연도별 시계열 형태로 압축 추출했습니다.
  * `ott_usage.csv` (이용 여부) : 4행 × 57,259열
  * `ott_time.csv` (이용 시간) : 4행 × 46,957열
  * `ott_money.csv` (지출 비용) : 3행 × 46,957열

## 3. 데이터 전처리 및 피처 엔지니어링 (Feature Engineering)

방대한 다차원 원시 데이터에서 머신러닝 모델 학습이 가능한 형태로 품질을 끌어올리기 위해 아래와 같은 핵심 전처리를 수행했습니다.

* **결측치(Missing Value) 및 노이즈 처리**:
  * 설문 미응답, 무효 응답 등으로 발생한 다량의 결측치를 단순히 제거하지 않고, **시계열 기반 피처 엔지니어링 파이프라인**을 통해 유의미한 변수로 변환 및 대체 가공했습니다.
* **시계열 데이터 집계 및 파생 변수 생성**:
  * 가입자별 `ott_money`, `ott_time`, `ott_usage` 데이터의 5개년 흐름을 분석하여, 단순 현재 상태가 아닌 **'이용 트렌드의 변화량'**을 포착할 수 있는 **48개의 고차원 파생 피처(Feature)**를 설계했습니다.

### 주요 생성 파생 변수 (48개 피처 엔지니어링 항목)
1. **기본 통계량:** 유저별 다개년 평균, 표준편차, 변동계수(CV)
2. **추세 및 변화:** 이용 추세(Trend) 기울기, 전년 대비 최근 변화량, 최대 증가폭, 변화량의 표준편차
3. **이탈 징후 지표:** 다개년 이용 감소 횟수, 연속 감소 횟수, 역대 최대 감소폭(Max Drop), 최근 2~3개년 연속 감소 여부 안전 장치(Safe Score), 최근 이탈 위험도 점수(Decline Score) 등

## 4. 최종 전처리 결과 및 데이터셋 분할

머신러닝 모델 학습을 위해 최종 가공 및 확장 변환이 완료된 데이터셋(`visualization_data.json` 연계)의 구성 및 분할 결과입니다.

### 4.1 데이터 볼륨 변화 요약
* **전처리 전:** 다개년 독립 설문 데이터 (총 2,637개 행, 연도별 약 10,000개 열의 차원 노이즈 존재)
* **전처리 및 시계열 확장 후:** 총 **11,040개 학습 샘플 데이터 행, 52개 정제 피처 열**로 변환 완료.
  * 5개년 패널 데이터의 연속 매칭 및 시계열 데이터 가공, 윈도우 슬라이딩 확장을 통해 인공지능이 충분한 패턴을 학습할 수 있는 볼륨으로 고도화되었습니다.

### 4.2 데이터셋 분할 (Dataset Split) 결과
모델의 과적합을 방지하고 일반화 성능을 투명하게 평가하기 위해 데이터셋을 **학습(Train) : 검증(Validation) : 시험(Test)** 데이터로 **60 : 20 : 20** 비율로 엄격하게 분할하였습니다.

| 데이터셋 구분 | 비율 (%) | 샘플 수 (행) | 비고 |
| :--- | :---: | :---: | :--- |
| **학습 데이터 (Train)** | **60%** | **6,624건** | AI 모델 패턴 학습용 데이터셋 |
| **검증 데이터 (Validation)** | **20%** | **2,208건** | 하이퍼파라미터 튜닝 및 학습 조기종료(Early Stopping) 검증용 |
| **시험 데이터 (Test)** | **20%** | **2,208건** | 모델 최종 성능 평가용 (혼동 행렬 검증 및 평가 완료) |
| **총합 (Total)** | **100%** | **11,040건** | **인공지능 모델링 최적화 데이터셋 완비** |

## 5. 최종 데이터 기반 AI 모델 성능 요약

가공된 전처리 데이터를 최적화 알고리즘(`XGBoost_Tuned`)에 적용하여 측정한 최종 성능 지표입니다. 데이터 전처리의 높은 신뢰성을 방증합니다.

* **정확도 (Accuracy):** Val 98.14% / **Test 98.46%**
* **정밀도 (Precision):** Val 89.32% / **Test 93.61%**
* **재현율 (Recall):** Val 92.89% / **Test 91.11%**
* **F1-Score:** Val 91.07% / **Test 92.34%**
* **AUC-ROC:** Val 0.9971 / **Test 0.9970**
* **특이사항:** 상위 10% 고위험군 가입자에 대한 예측 정밀도(Precision@10) **100.0% 달성**

## 6. 결론 및 기대 의의
1. **차원의 저주 완벽 해결:** 연도별 1만 개에 달하던 방대한 설문 변수를 OTT 이탈 예측에 최적화된 핵심 시계열 중심 데이터로 압축하여 모델 복잡도를 제거했습니다.
2. **이탈 예측 성능 극대화:** 단순 데이터 매칭에 그치지 않고, 이용 패턴의 감소 지속성 및 변동계수 등 고도화된 48개 파생 변수를 제공함으로써, 최종 모델이 **테스트 데이터 기준 98.46%의 압도적인 정확도와 0.9970의 AUC-ROC**를 확보할 수 있는 고품질 데이터 기반을 증명했습니다.
